# config-afterglow

> Figma Config 2026 の余韻。71本のうち、あなたに残る一本を。

Figma Config 2026 の公開セッション動画（71本）から、興味・気分・AIとの距離で「あなたに残る一本」を見つける、自己完結型の Web ページ。

**公開URL**: （GitHub Pages 有効化後にここへ）

## これは何

- 71本のセッションを、登壇者が実際に壇上で話した内容から書き起こした「感触（the feeling）」とタグで並べる
- 湿度（dry ↔ wet）・呼応するペア・「AIの話ではなかった一本」など、いくつかの切り口で辿れる
- 「Pull one for me」でおみくじのように一本引ける
