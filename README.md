# Config Afterglow

> Figma's Config 2026 — its 71 public talks, made into a place you can wander.

**Live:** https://littlethelily.github.io/config-afterglow/

An unofficial, self-contained web piece built from the 71 public talks of Figma Config 2026. Pick how you're feeling, and let a few of them find you — or pull one at random. Everything links back to its original on YouTube.

## What it is

- Each talk carries a short, hand-written *the feeling* — my own reflection, written after watching, from what each speaker actually said on stage. Quotation marks mark their words; the rest is mine.
- You wander by **humidity** (dry reason ↔ wet feeling), through **pairs** that answer each other, or into **Still, craft** — the quiet talks that never mention AI.
- **Pull one for me** hands you a single talk at random, like drawing a fortune.
- Japanese / English throughout.

The idea underneath: Config 2026 wasn't all about AI, and it wasn't all about Figma. A care for making by hand, for what lasts, was there in the same room. This is the afterglow of that.

## Further reads (included)

- `config2026_agentic_workflows_notes.html` — hand-drawn viewing notes on one talk, "Agentic workflows and the MCP"
- `config2026_cursor_demo_notes.html` — notes from a Cursor demo night, a Config side event

## How it's made

A single self-contained HTML page — fonts, images, CSS and JS all embedded, no build step. The whole production ran alongside AI (subtitle-checking, parallel generation, a data pipeline), so that *how* it was made would match *what* it's about. The discipline was to cut, not to add: of the axes I tried, only the ones that actually helped you narrow down were kept.
